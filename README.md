# Portfolio

## Deploy (Render)

This project is ready to deploy as a Node web service (Express + Vite build output).

1. Push this repo to GitHub.
2. Go to https://render.com and click `New` -> `Blueprint`.
3. Select this GitHub repository.
4. Render will detect `render.yaml` automatically.
5. Click `Apply`.

After deployment finishes, Render will provide a public URL like:
`https://pratik-portfolio.onrender.com`

Use that URL as your shareable portfolio link.
